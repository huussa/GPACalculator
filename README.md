# GPA Calculator

A simple web application for calculating semester and cumulative GPA.

## Features

* Calculate semester GPA
* Calculate cumulative GPA
* Add courses dynamically
* Edit existing courses
* Delete courses
* Save data locally using Local Storage

## Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation)

## Live Demo

Website: https://huussa.github.io/GPACalculator/

## How to Use

1. Add your courses and credit hours.
2. Select the grade for each course.
3. The GPA will be calculated automatically.
4. Your data is saved locally in the browser.
